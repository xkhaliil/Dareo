🎮 Dareo
📌 Project Description

Dareo is a gamified social web application where friends create private groups and challenge each other with dares.

Each dare can either be directly assigned to a specific player or opened for group voting to decide who must complete it. Players earn XP for successfully completing dares, level up over time, and compete on a leaderboard.

The platform combines social interaction, competition, and game-style progression mechanics in a dynamic and animated user interface.

Dareo focuses on:

Friendly competition

Group interaction

Gamification (XP, levels, ranks)

Engaging game-inspired UI

🚀 Core Features (MVP)

👥 Create and join private groups

🎯 Create dares for group members

🗳 Assign dares directly or allow group voting

⏳ Voting phase with automatic assignment

✅ Mark dares as completed

🗳 Group validation through voting

⭐ XP reward system

📊 Level progression system

🏆 Leaderboard ranking within groups

🎮 Game-style UI with animations

🎮 Game Mechanics

Players earn XP for completing dares.

XP determines level progression.

Levels unlock new ranks.

Leaderboard updates dynamically.

Bonus XP may be awarded for voted dares.
