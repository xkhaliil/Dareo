# components/ui/

This directory is managed by **shadcn/ui** (`npx shadcn add <component>`).

- Do **not** edit files here manually — they are generated primitives.
- Do **not** add your own components here — use `src/features/<domain>/components/` or `src/shared/components/` instead.
- The path `@/lib/utils` (in `src/lib/utils.ts`) is required by shadcn's generated code and must remain.

All application-level shared components live in `src/shared/components/`.
